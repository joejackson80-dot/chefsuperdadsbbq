# Chef SuperDad's BBQ & Catering

North Omaha's Premier Full-Service BBQ Catering website. Built for Pitmaster Cameron Lollar.

## Features
- **Fire-Inspired Aesthetics**: Custom dark smoky theme with ember effects.
- **Ember Particle Engine**: Live JavaScript-powered ember animation in the hero section.
- **Detailed Menu**: Complete listing of Bulk Meats, Individual Meats, and Side Items with current pricing.
- **Quote Request System**: Comprehensive lead generation form for custom event catering.
- **Mobile Responsive**: Fully optimized for all screen sizes.

## Local Development
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. To customize images, update the files in the `images/` directory.

## Deployment
This is a static site. It can be deployed via GitHub Pages, Netlify, or Vercel by simply connecting this repository.
